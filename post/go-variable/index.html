Go 变量声明

2022-05-26
Go
times read

Contents

静态语言区别于动态语言的一个特征：变量声明。

为了方便操作内存特定位置的数据，我们用一个特定的名字与位于特定位置的内存块绑定在一起，这个名字就是：变量。

变量所绑定的内存区域要有一个明确的边界，边界由 “变量声明” 确定。

使用变量之前，需要先进行变量声明。

单独声明：

1

 var a int = 10

默认原生类型的默认值：

内置原生类型 默认值 (零值)
所有整型类型 0
浮点类型 0.0
布尔类型 FALSE
指针、接口、切片、channel、map和函数类型 nil

变量声明块：

1
2

var (
		a int = 128
		b int = 120
		s string = "heool"
		c rune = 'A'
		t bool = true
)

一行变量声明中同事声明多个变量：

1

var a, b, c int = 1, 2, 33

同样的，也可在变量声明块中使用多变量声明：

1
2
3
4

var (
		a, b, c int = 1, 2, 3
		c, d, e rune = 'A', 'D', 'E'
)

两种变量声明 “语法糖”

  1. 省略变量声明中的类型信息。必须显式地赋予初始值。编译器通过给的初始值，自动推导出类型信息。

    1
2
3

    var b = 12 // 使用默认的类型
var c = int32(99) // 显式类型转换
var d, e, f = 12, 'A', "hello" // 多变量声明

  2. 短变量声明。

    1
2
3
4
5
6
7

    // 单独变量声明
a := 12
b := 'A'
c := "hello"

// 多变量声明
a, b, c := 12, 'A', "hello"

变量的作用范围：

  1. 包级变量

    包级别可见的变量。如果是导出变量(大写字母开头)，那么可被视为全局变量。

    声明方式，只能使用带有 var 关键字的变量声明方式，不能使用短变量声明形式。

var a = int32(123)

  	// use var a
}

    Go 函数或方法体内声明的变量，仅在函数或方法体内可见。

Go 和 C 声明方式上的不同：

Go 是从左至右的读法，C 是螺旋式的读法。
